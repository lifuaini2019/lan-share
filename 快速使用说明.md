# 🚀 祖宇字文共享 v2.1.0 - 快速使用说明

## 📱 玩客云/路由器用户

### 一键安装（推荐）
```bash
# SSH连接到玩客云后执行：
curl -fsSL https://raw.githubusercontent.com/lifuaini2019/lan-share/main/install_wankeyun.sh | bash
```

### 手动安装
```bash
# 1. 下载脚本
wget https://raw.githubusercontent.com/lifuaini2019/lan-share/main/install_wankeyun.sh

# 2. 运行安装
chmod +x install_wankeyun.sh
./install_wankeyun.sh
```

**适用设备**: 玩客云、小米路由器、OpenWrt路由器等 ARMv7 设备

## 💻 Windows/macOS/Linux用户

### 直接下载运行
1. 前往 [GitHub Releases](https://github.com/lifuaini2019/lan-share/releases/latest)
2. 下载对应平台的可执行文件
3. 解压后直接运行
4. 浏览器打开 `http://localhost:9405`

### 支持的文件
- **Windows 64位**: `zuyu-share-windows-amd64.exe`
- **Windows 32位**: `zuyu-share-windows-386.exe`
- **macOS Intel**: `zuyu-share-darwin-amd64`
- **macOS Apple Silicon**: `zuyu-share-darwin-arm64`
- **Linux 64位**: `zuyu-share-linux-amd64`

## 🌟 新功能: 智能局域网检测

### 🎯 工作原理
- 当您通过域名访问程序时，系统会自动检测您是否在局域网环境
- 如果检测到在局域网，会弹出美观的提示框询问是否切换到局域网地址
- 切换后可以获得更快的访问速度和更稳定的连接

### 🎨 用户体验
- 智能判断，不会打扰正常使用
- 美观的现代化提示界面
- 一键切换，1秒内完成跳转
- 提供详细的网络信息展示

## 🔧 常用管理命令

### 玩客云/路由器
```bash
# 服务管理
systemctl start zuyu-share      # 启动
systemctl stop zuyu-share       # 停止
systemctl restart zuyu-share    # 重启
systemctl status zuyu-share     # 状态

# 查看日志
journalctl -u zuyu-share -f
```

### 飞牛OS Docker版
```bash
# 容器管理
docker start zuyu-share      # 启动
docker stop zuyu-share       # 停止  
docker restart zuyu-share    # 重启
docker ps | grep zuyu-share  # 状态

# 查看日志
docker logs zuyu-share -f
```

## 🆘 常见问题

### Q: 安装失败怎么办？
**A**: 检查网络连接和GitHub访问，确保可以访问 `github.com`

### Q: 服务无法启动？
**A**: 检查端口9405是否被占用：`netstat -ln | grep 9405`

### Q: 无法访问页面？
**A**: 检查防火墙设置，确保端口9405开放

### Q: 智能检测不工作？
**A**: 确保在局域网环境下通过域名访问，并查看浏览器控制台日志
