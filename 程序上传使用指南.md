## 🖥️ 各平台上传和部署指南

### 📱 玩客云/路由器部署

#### 方式1: 一键安装（推荐）
```bash
# SSH连接到玩客云后执行
curl -fsSL https://raw.githubusercontent.com/lifuaini2019/lan-share/main/install_wankeyun.sh | sudo bash
```

#### 方式2: 手动上传部署
```bash
# 1. 在电脑上下载程序
wget https://github.com/lifuaini2019/lan-share/raw/main/build/zuyu-share-linux-arm

# 2. 使用scp上传到玩客云
scp zuyu-share-linux-arm root@玩客云IP:/opt/

# 3. SSH连接到玩客云
ssh root@玩客云IP

# 4. 设置权限并运行
chmod +x /opt/zuyu-share-linux-arm
cd /opt
./zuyu-share-linux-arm
```

### 🖥️ 飞牛OS部署

#### 方式1: 一键安装（推荐）
```bash
# SSH连接到飞牛OS后执行
curl -fsSL https://raw.githubusercontent.com/lifuaini2019/lan-share/main/install_flynas.sh | sudo bash
```

#### 方式2: Docker镜像部署
```bash
# 1. 下载Docker镜像到本地
wget https://github.com/lifuaini2019/lan-share/raw/main/zuyu-share-flynas-arm64.tar

# 2. 上传到飞牛OS（使用飞牛OS文件管理器或scp）

# 3. SSH连接到飞牛OS
ssh root@飞牛OS_IP

# 4. 导入并运行Docker镜像
cd /tmp
docker load -i zuyu-share-flynas-arm64.tar
docker run -d --name zuyu-share --restart unless-stopped -p 9405:9405 zuyu-share:latest
```

## 🔧 高级部署选项

### 系统服务化部署

#### Linux系统 (systemd)
```bash
# 1. 创建服务文件
sudo tee /etc/systemd/system/zuyu-share.service > /dev/null << EOF
[Unit]
Description=Zuyu Share Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/zuyu-share
ExecStart=/opt/zuyu-share/zuyu-share-linux-amd64
Restart=always
RestartSec=3

[Install]
WantedBy=multi-user.target
EOF

# 2. 启用并启动服务
sudo systemctl daemon-reload
sudo systemctl enable zuyu-share
sudo systemctl start zuyu-share

# 3. 查看服务状态
sudo systemctl status zuyu-share
```

### Docker Compose部署
```yaml
# docker-compose.yml
version: '3.8'
services:
  zuyu-share:
    image: zuyu-share:latest
    container_name: zuyu-share
    restart: unless-stopped
    ports:
      - "9405:9405"
    volumes:
      - ./data:/app/data
    environment:
      - TZ=Asia/Shanghai
```

## 🌐 网络配置

### 防火墙设置
```bash
# CentOS/RHEL
sudo firewall-cmd --permanent --add-port=9405/tcp
sudo firewall-cmd --reload

# Ubuntu/Debian  
sudo ufw allow 9405/tcp

# 通用iptables
sudo iptables -A INPUT -p tcp --dport 9405 -j ACCEPT
```

### 域名绑定
```bash
# 如果有域名，可以通过反向代理访问
# Nginx配置示例
server {

    location / {

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;

}
```

## 🆘 故障排除

### 常见问题

**1. 程序无法运行**
```bash
# 检查文件权限
ls -la zuyu-share-*
chmod +x zuyu-share-*

# 检查系统架构
uname -m
```

**2. 端口被占用**
```bash
# 查看端口占用
netstat -ln | grep 9405
lsof -i :9405

# 杀死占用进程
sudo kill -9 进程ID
```

**3. 无法访问页面**
```bash
# 检查程序是否运行
ps aux | grep zuyu-share

# 检查防火墙
sudo ufw status
```

**4. 智能检测不工作**
- 确保在局域网环境下通过域名访问
- 检查浏览器控制台是否有错误
- 访问 `/test-lan` 页面进行功能测试
